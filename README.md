# PRO-FIT
졸업프로젝트    
옷을 3D, 가상으로 피팅해볼 수 있는 안드로이드 어플리케이션 서비스
    
## 주요 기능
- 신체 측정
- 가상 옷 피팅
- 결과 영상 다운로드
- 옷 등록
    
## 특징
- 동영상 기반 가상 피팅으로 사용자가 뒷모습까지 확인할 수 있습니다.
- 사용자 신체 사이즈가 자동으로 측정됩니다.
- 별도의 센서 없이 동영상 기반의 영상처리를 진행합니다
- 사용자가 가상 피팅 뿐만 아니라 옷 등록이 가능합니다.
